# Trabajo de Investigación Formativa - Cálculo Integral

Método numérico utilizado: Regla de Simpson 1/3 compuesta.

Aplicación: Volumen de un sólido de revolución mediante arandelas.

El programa calcula una integral definida usando Simpson 1/3, aproxima el volumen del sólido de revolución y compara el resultado numérico con el valor exacto cuando está disponible.
